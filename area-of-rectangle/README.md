📐 Area of Rectangle

A beginner React app that calculates the area of a rectangle based on user-provided length and width.

✨ Features

Input fields for length and width
Instant area calculation on input change
Clean, minimal UI

📦 Run Locally
bashcd area-of-rectangle
npm install
npm start

💡 Concepts Covered

useState for controlled inputs
Arithmetic operations in JSX
Event handling with onChange

🧮 Formula
Area = Length × Width
